# 世界の民主度・自由度マップ (World Freedom Map)

民主度・言論の自由度・人権・法の支配・腐敗・紛争死者数など9つの指標で世界地図を色分けするアプリ。
「戦争・紛争を起こすのは政権交代のない独裁国家が多いのでは」という仮説を、
民主度マップに紛争死者数の円を重ねて視覚的に検証できる。

## 起動

このフォルダで:

```
py -m http.server 8590
```

→ http://localhost:8590/

（サーバーなしで index.html をダブルクリックしても動く。データは全てローカルJS化済み）

## 機能

- 指標セレクタで9指標を切り替え（V-Dem各指数、政治体制4分類、腐敗認識指数、紛争死者数×2）
- 0〜1指数は赤（独裁的）↔青（民主的）のダイバージング配色、紛争死者数は赤の対数スケール
- 「紛争死者数を重ねる」チェックで、10年累計死者数を円の大きさで民主度マップ等に重畳
- ホバーでツールチップ（値・順位・年）、クリックで全指標の詳細パネル
- ランキング一覧（検索付き）、ホイールズーム＆ドラッグパン、ダークモード対応

## データ

- `data/*.csv` … Our World in Data の grapher CSV（V-Dem、Regimes of the World、
  Transparency International CPI、UCDP/PRIO 紛争死者数）
- `data/countries.geo.json` … 世界地図 GeoJSON（johan/world.geo.json、ISO3キー）
- `build_data.py` … CSVから各国の最新値を抽出し `data/indicators.js` と `data/geo.js` を生成

データ更新時は CSV を再ダウンロードして `py build_data.py` を実行。

## 注意

- EIU民主主義指数は再配布不可（OWIDが403を返す）ため、V-Dem指数で代替
- 紛争死者数は「その国の領域内で起きた紛争」の死者数（侵略した側ではなく戦場の所在地に計上される）
- 2014年より前が「最新値」の国（消滅国家等）は表示から除外
